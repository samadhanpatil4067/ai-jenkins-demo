# 🚀 AI-Enabled Jenkins Pipeline (2026)

This project demonstrates how to integrate AI into Jenkins pipelines for:

- Failure analysis
- Root cause detection
- Intelligent DevOps automation

## 🧪 Demo Flow

1. Code fails intentionally
2. Jenkins captures logs
3. AI analyzes failure
4. Provides root cause & fix

## ⚙️ Setup

- Jenkins (Docker recommended)
- Python 3
- OpenAI API key

## 🔐 Configure API Key in Jenkins

Add credential:
- Kind: Secret Text
- ID: openai-api-key

## ▶️ Run

Trigger Jenkins pipeline job
